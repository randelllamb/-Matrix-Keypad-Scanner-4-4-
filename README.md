# Matrix Keypad Scanner (4×4)

This project implements scanning and key detection for a 4×4 matrix keypad using the Tiva LaunchPad. It identifies pressed keys by scanning rows and columns.

## Features
- Row/column scanning algorithm  
- Key press detection  
- Debouncing logic  
- ASCII key output  

## Skills Demonstrated
- Matrix keypad interfacing  
- Digital input handling  
- Debounce filtering  
- Embedded systems programming  

## Hardware
- Tiva C LaunchPad  
- 4×4 Matrix Keypad  
- Breadboard & wiring
